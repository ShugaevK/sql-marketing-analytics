# Краткое резюме: SQL для маркетинга

## Цель кейса
Этот кейс предназначен для демонстрации практических SQL знаний в контексте маркетинг аналитики.

## Ключевые компетенции

- **Группировка данных**: GROUP BY, DATE_TRUNC
- **Агрегатные функции**: COUNT, SUM, AVG, ROUND
- **Объединение таблиц**: JOIN, LEFT JOIN
- **Условные операторы**: CASE WHEN, NULLIF, HAVING
- **Подзапросы**: WITH (CTE)

## Метрики в кейсе

| Метрика | Описание | Формула |
|---|---|---|
| **CTR** | Процент переходов | клики / показы * 100 |
| **CPC** | стоимость клика | расходы / клики |
| **CPA** | стоимость привлеченных | расходы / конверсии |
| **ROAS** | отношение дохода | доход / расходы |
| **Conversion Rate** | процент конверсий | конверсии / сессии * 100 |
